◆ is_strict_dep_MEM()

bool Instruction::is_strict_dep_MEM ( Instruction *  i2)

conservatively test if there is a potential MEMDEP dependence between the current instruction and i2

Returns
return true if the current instruction and i2 are two memory accesses different from two reads when accesses are not related to the stack pointer. In case of memory accesses, different from two reads, relative to the stack pointer, the returned value depends on the immediate offsets.
